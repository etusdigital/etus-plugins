# ⚠️ Deprecated Command

This command has been deprecated in ids.yml v3.

**Replacements:** Individual data commands available:
- `/data-reqs` — Data requirements
- `/erd` — Entity relationship diagram
- `/db-spec` — Combined database requirements and schema
- `/dictionary` — Data dictionary
- `/dfd` — Data flow diagram
- `/catalog` — Data catalog

**Reason:** Grouped command eliminated in favor of individual command flexibility.
